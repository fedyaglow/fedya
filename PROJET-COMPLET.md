## 🚀 DÉMARRAGE RAPIDE

### 1️⃣ Configuration (2 minutes)

**Modifier le numéro WhatsApp:**
```javascript
// Dans assets/js/whatsapp.js ligne 11
this.phoneNumber = 'VOTRE_NUMERO'; // Ex: 212612345678
```

**Format:** Code pays + numéro (sans +, sans espaces)

### 2️⃣ Test Local (1 minute)

Double-cliquez sur `index.html` → Le site s'ouvre dans votre navigateur!

### 3️⃣ Mise en Ligne (2 minutes)

**Option A - Netlify (Le plus simple):**
1. Allez sur https://app.netlify.com/drop
2. Glissez-déposez le dossier `Fedya-Glow`
3. ✅ Site en ligne!

**Option B - GitHub Pages:**
```bash
cd Fedya-Glow
git init
git add .
git commit -m "Mon site Fedya Glow"
git push origin main
```
